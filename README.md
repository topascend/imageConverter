# imageConverter
## 图片格式转换



### 支持jpg、jpeg、png、gif、bmp、tiff图片文件的互相转换

### 命令行运行
![图片格式转换](https://github.com/topascend/imageConverter/blob/main/demo/2.jpg)
```azure
.\imageConverter.exe -dir E:\cmd\img -ext png -saveDir testimg
```
